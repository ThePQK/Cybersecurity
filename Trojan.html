Trojan

A trojan is a malware that disguises itself as a beign application and it will reside in the infected computer without showing indications it already exists in the computer.



How Trojans Function

Trojan neither self-replicate nor attach to host programs to replicate. It requires human interaction in order to spread. The victim has to download the trojan in order for it to infect his computer. This is done by disguising the trojan as trustworthy application. Once the appplication has been downloaded and runned, the trojan will remain in the computer without doing anything. The victim will suspect nothing and will be worried-free of the application downloaded. The trojan will start to execute its malicious code until a specific date or a specific actions such as the victim entering his bank details on an online bank website.

What Can Trojans Do ?

1. Install backdoors for hackers to have remote access to the infected computer

2. Install malware (E.g.: Virus, Worms) on the infected computer

3. Keystroke logging

4. Activate the computer's camera and start recording what can be seen by the camera
